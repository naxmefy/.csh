# .csh

my custom shell configuration, extension, etc.

## installation

- clone `git clone git@github.com:naxmefy/.csh.git ~/.csh`
- add following to `.bashrc`/`.zshrc`: ``` source ~/.csh/.cshrc```

```bash

# bash
echo "\nif [ -f ~/.csh/.cshrc ]; then\n\tsource ~/.csh/.cshrc\nfi\n" >> .zshrc

# zsh
echo "\nif [ -f ~/.csh/.cshrc ]; then\n\tsource ~/.csh/.cshrc\nfi\n" >> .zshrc

```
